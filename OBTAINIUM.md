# Obtainium

Easily install and keep **TurtlPass** up to date on Android by tracking official GitHub releases with **Obtainium**.

---

## Install Obtainium

1. **Download**  
   Get the latest Obtainium APK from the official GitHub releases page:  
   [https://github.com/ImranR98/Obtainium/releases](https://github.com/ImranR98/Obtainium/releases)

2. **Install**  
   Install the downloaded APK on your Android device.  
   After installation, Obtainium can update itself directly within the app.

---

## Install TurtlPass via Obtainium

1. **Copy the repository URL**

   ```
   https://github.com/TurtlPass/turtlpass-android
   ```

2. **Add the app in Obtainium**

   * Open **Obtainium**
   * Tap **“Add App”**
   * Paste the URL into **“App Source URL”**
   * Leave the default settings unchanged
   * Tap **“Add”**

3. **Install TurtlPass**  
   You'll be taken to the app’s information page. Tap **“Install”** to begin.

---

Obtainium will automatically check for new TurtlPass releases and notify you when updates are available.
